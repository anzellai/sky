# Go Stdlib

Demonstrates using Go standard library packages from Sky, including `crypto/sha256`, `encoding/hex`, `time`, and `net/http`.

## Build & Run

```bash
sky install
sky build src/Main.sky
./sky-out/app
```

Requires `sky install` first to generate FFI bindings for Go stdlib packages.
