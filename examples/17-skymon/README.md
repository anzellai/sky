# SkyMon

Sky.Live monitoring dashboard with uptime monitors, custom metrics, alerting, and SVG charts. 20 modules, approximately 3300 lines of pure Sky code with proper ADT types throughout.

## Build & Run

```bash
sky install
sky build src/Main.sky
./sky-out/app
```

Open `http://localhost:8000` in your browser. Configure monitors and alert thresholds from the dashboard.
